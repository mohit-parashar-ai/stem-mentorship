## 🔍 Troubleshooting

### Logo Not Showing?

**Check 1: File exists**
```bash
ls -lh public/foi-logo.png
```

**Check 2: Clear browser cache**
Press: Ctrl/Cmd + Shift + R

**Check 3: Restart dev server**
```bash
# Stop: Ctrl + C
# Start: npm run dev
```

### Logo Looks Blurry?

This is normal if viewing on a high-DPI (Retina) display with a low-resolution source image. Consider:
- Using a higher resolution source (512×512 or 1024×1024)
- Converting to SVG for perfect scaling
